2021 Computers in Cardiology Competition Submission

Two dimensional models

In the provided two dimensional codes, you can interact with the simulations by clicking in the visulaization area, and through the graphical user interface.

Through the graphical user interface, you can

  • change model parameters;
  • setup pacemakers;
  • change the behaviour of the mouse click;
  • alter visualization settings;
  • Edit the source code of different shaders during the simulations;
  • Save and reload simulation through json files;

Be bold and make changes! If things don't work out, you can always reload the page for a fresh start!

Choose any of the following codes to start:

  1. Three variable minimal model
  2. Excitations in the CinC (LOGO?!)
  3. The 41-Parameter OVVR Model in 2D

Three dimensional models

For running the three dimensional codes, you need to download the structure files and load them into the WebGL program. After downloading the file, unzip it and choose the file with the desired resolution to run the interactive simulations.

Similar to the 2D codes, you can interact with the 3D codes through the graphical interface and the visualization area.

To create excitations, use Ctrl+Drag in Microsoft Windows and Linux, and Command+Drag in macOS. To choose the voltage trace location, use Shift+Click on the visualized structure.

  1. Three variable minimal model
  2. The OVVR model with TNNP sodium dynamics

Download the structure files before you start any 3D simulation!

You can download the source code from "https://github.com/kaboudian/CinC-2021-Competition" git repository.
